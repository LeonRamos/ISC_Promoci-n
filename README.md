### Cómo personalizar
- Para cambiar el texto del badge (por ejemplo, poner versión), edita la URL:
  `https://img.shields.io/badge/Nombre-VALOR?logo=logoName&logoColor=ffffff`
- Para cambiar el color: reemplaza el valor hexadecimal tras el `-` (por ejemplo `-FF5733`).
- Para ocultar el enlace, elimina la parte `(https://...)` que envuelve la imagen.
